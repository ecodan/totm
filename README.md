# TOTM: Theater of the Mind

Theater of the Mind (TOTM) is a project developed using the Cicadas SDD methodology. It creates a platform for playing role playinggames with AI agents.

## Getting Started

1. **Bootstrap**: Run `python scripts/chorus/scripts/init.py` (if not already done).
2. **Status**: Run `python scripts/chorus/scripts/status.py`.
3. **Migrate**: Run `python migrate_cicadas.py` to sync with upstream Cicadas.
